## Document Scoring and Inter-Rater Reliability

Relevance scoring was performed using two large language model (LLM) scorers operating in parallel: Anthropic Claude Haiku (`claude-haiku-4-5-20251001`) and a Codex/GPT local-agent scoring run (`gpt-5.4-mini-local-agents`). Both scorers applied the same four-dimension rubric: care-first governance relevance, AI/technology governance relevance, intersection between the two governance systems, and evidentiary quality. Composite scores were computed locally using the fixed formula: `composite = carefirst*0.25 + ai_governance*0.25 + intersection*0.35 + evidentiary*0.15`.

Claude scored 7,425 documents and Codex/GPT scored 7,265 documents, with 6,988 shared documents matched by SHA-256 content hash. After Codex-side reliability correction, raw independent Claude-vs-GPT agreement remained limited: Cohen's kappa = 0.1327, quadratic-weighted kappa = 0.2500, ICC(2,1) = 0.2914, Krippendorff alpha = 0.2652, Pearson r = 0.3222, and exact tier agreement = 48.7%. These raw IRR values are disclosed as a limitation.

Final tiers are based on resolved scores, not on either model alone. A **resolved tier** is the tier assigned after combining both scorers' sub-scores. Agreement with the resolved tier reaches the target threshold: Cohen's kappa = 0.5271 for Claude vs averaged resolved tier, and Cohen's kappa = 0.5164 for Claude vs final clipped resolved tier. These resolved-tier kappas are not independent IRR; they measure how closely a scorer aligns with the final classification used for the review set.

The final merged corpus contains 7,702 documents. After the legacy post-merge intersection ceiling, the final review set contains 923 Tier 1+2 documents, including 9 robust Tier 1 documents that both scorers independently classified as Tier 1. Documents with an absolute Claude-vs-GPT composite difference greater than 3.0 remain flagged as contested (n = 654 total; n = 256 in the final Tier 1+2 review set) and should receive human review before being used as primary evidence.
